## 📁 仓库结构建议

```bash
hands-on-ml3/
├── README.md                # 仓库说明
```

## ✅ 使用建议

- 每章使用一个 `.ipynb` 文件，内容包括：
  - 📖 代码实现
  - 🧠 注释说明
  - 📝 自我总结（写在 markdown 单元格中）

- 每完成一章，将 notebook 上传至 GitHub：
  - 菜单栏：**文件 → 保存副本到 GitHub**
  - 填写清晰的文件名和提交说明
